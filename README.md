## Description

Estudo Arquitetura hexagonal, teste unitários e teste de carga Pós FIAP Software Architecture.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Documentation Swagger

[clique aqui](http://localhost:3002/api)

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
